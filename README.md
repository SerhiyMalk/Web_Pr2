# Система моніторингу КРМ 

Веб-додаток для моніторингу параметрів Компенсаторної установки (КРМ) у реальному часі. Проєкт реалізовано як Progressive Web App (PWA) з використанням сучасних технологій фронтенд-розробки. 

**Варіант 11:** Компенсаторна установка (КРМ)

## Основні можливості

- **Моніторинг у реальному часі:** Імітація потокового передавання даних через WebSocket.
- **Візуалізація даних:** Побудова динамічних графіків зміни реактивної потужності та напруги (за допомогою `Chart.js`).
- **Аналіз трендів:** Автоматичне визначення напрямку зміни показників (зростання ↗️, спадання ↘️, стабільність ➡️).
- **Система сповіщень:** Інтеграція з Browser Notifications API для надсилання push-повідомлень при критичних відхиленнях параметрів.
- **PWA (Progressive Web App):** Можливість встановлення додатку на пристрій та підтримка офлайн-режиму (Service Workers).
- **Сучасний UI/UX:** Адаптивний дизайн (Grid/Flexbox), ефект Glassmorphism, підтримка світлої та темної тем оформлення.

## Технологічний стек

- **HTML5:** Семантична розмітка.
- **CSS3:** Кастомні змінні, CSS Grid, анімації `keyframes`, медіа-запити.
- **JavaScript (ES6+):** Асинхронна робота, робота з DOM, обробка подій.
- **Chart.js:** Бібліотека для малювання інтерактивних графіків у `<canvas>`.
- **PWA:** `manifest.json` та `sw.js` (Service Worker).

## Структура проєкту

├── index.html      # Головна сторінка та каркас додатку
├── style.css       # Стилі, теми та адаптивність
├── app.js          # Основна бізнес-логіка, WebSocket, тренди та графіки
├── manifest.json   # Конфігурація для PWA
└── sw.js           # Service Worker для кешування ресурсів

##  Параметри моніторингу (Варіант 11)

- **Реактивна потужність:** -500 ... +500 кВАр (Норма: -100 ... 100)
- **Коефіцієнт потужності (cos φ):** 0.6 ... 1.0 (Норма: 0.92 ... 0.98)
- **Напруга мережі:** 350 ... 420 В (Норма: 380 ... 400)
- **Кількість увімкнених ступенів:** 0 ... 12 (Норма: 2 ... 8)
- **Температура конденсаторів:** 10 ... 80 °C
- **Стан контакторів:** 12 шт (Увімкнено/Вимкнено)
